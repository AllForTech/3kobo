# Contributing to 3Kobo

We welcome contributions from:
- Educators
- Linguists
- Developers
- Students
- Researchers
- GSoC Contributors

## Ways to Contribute

- Contribute to the Core Codebase
- Submit new vocabulary terms
- Improve translations
- Review and validate datasets
- Improve documentation
- Build integrations or tools

## Contribution Principles

- Respect linguistic and cultural accuracy
- Follow ethical and inclusive practices
- Attribute sources where applicable

All contributions are reviewed by language experts and maintainers before acceptance.
